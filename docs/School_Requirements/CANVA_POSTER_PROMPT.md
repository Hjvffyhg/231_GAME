## 🎨 Layout & Design Instructions for Canva Editor
*Use this structure to divide your 10x15 poster into clean, scannable sections. Use text boxes in Canva and copy-paste the content below.*

### 1. Header Section (Top)
*   **Main Title:** SPACE SURVIVAL (Large, bold, tech/sci-fi font, e.g., *Space Grotesk* or *Orbitron*)
*   **Subtitle:** Earth Defense Initiative — Tactical Action Meets OS Scheduling
*   **Visual:** Place a screenshot of gameplay or the Main Menu background here as a faded backdrop.

### 2. Project Overview & Objectives
*   **Overview:** A top-down survival game that integrates operating system CPU scheduling concepts into a tactile combat mechanic. Command a fleet of automated drones to defend against the Kla'ed armada.
*   **Objectives:** 
    * To educate players on computer science architectures.
    * To visually demonstrate CPU scheduling algorithms' efficiency, prioritization logic, and waiting times in a real-time combat simulation.

### 3. Key Features (Grid or Bullet Points)
*Include small icons next to each point:*
*   🖥️ **Tactical UI:** Immersive, terminal-style Heads-Up Display with real-time telemetry.
*   ⚙️ **AI Autopilot:** Hands-free algorithm tester for observing logic without playing.
*   🛠️ **Hangar Progression:** Upgrade system to enhance ship chassis capabilities utilizing combat credits.
*   📖 **Database Codex:** Detailed lore, threat logs, and tactical metrics.

### 4. Gameplay & System Process (Flowchart Layout)
*Use Canva's flowchart/arrow shapes to connect these steps:*
1.  **DEPLOY:** Select threat sector via Tactical Starchart.
2.  **ENGAGE:** Maneuver the ship while drones auto-fire based on active OS Algorithm queues.
3.  **ADAPT:** Cycle scheduling modes (FCFS, RR, HRRN) dynamically to handle varying swarm compositions.
4.  **ANALYZE:** Review the Mission Report for algorithm efficiency and allocate upgrades.

### 5. OS Algorithm Integration (Core Requirement Panel)
*Make this the most prominent box on the poster to satisfy the academic rubric.*
*   **FCFS (First-Come, First-Served):** Attacks the oldest detected threat in the hazard zone. Susceptible to the "convoy effect."
*   **Round Robin (RR):** Rotates fire equally across the entire enemy cluster using time quantums. Excellent damage spread.
*   **HRRN (Highest Response Ratio Next):** Dynamic targeting algorithm prioritizing targets based on how long they've been alive (wait time) combined with their threat weight.
